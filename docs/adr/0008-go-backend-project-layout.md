# 0008 - Go Backend Project Layout

## Status

Accepted

## Context

Go backend layout applies to Modes B and C.

## Decision

No Go backend is scaffolded in Mode A.

## Consequences

There are no `cmd/`, `internal/`, `pkg/`, or Docker backend files.

## Alternatives Considered

A placeholder Go service was rejected because it would imply a runtime backend that v1 does not need.
